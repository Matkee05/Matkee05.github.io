Osnovni HTML tagovi

HTML koristi oznake kao što su <h1> za naslove, <p> za paragrafe, itd.

            <h1>Ovo je glavni naslov</h1>
            <p>Ovo je paragraf teksta</p>

Osnovni CSS stilovi

CSS se koristi za stilizovanje elemenata. Na primer:

            h1 { 
                color: blue; 
                font-size: 30px;
            }
            p {
                color: gray;
                font-family: Arial, sans-serif;
            }
