Tutorijal o HTML-u i CSS-u

Ovaj tutorijal vas vodi kroz osnovne HTML i CSS tagove i stilove.

Osnovni HTML tagovi

HTML koristi oznake kao što su <h1> za naslove, <p> za paragrafe, itd.

Primer HTML tagova:

            <h1>Ovo je glavni naslov</h1>
            <p>Ovo je paragraf teksta</p>

Osnovni CSS stilovi

CSS se koristi za stilizovanje elemenata. Na primer:

            h1 { 
                color: blue; 
                font-size: 30px;
            }
            p {
                color: gray;
                font-family: Arial, sans-serif;
            }

HTML Forms

HTML forme se koriste za unos podataka. Evo kako možete napraviti osnovnu formu:

            <form>
                <label for="name">Ime:</label>
                <input type="text" id="name" name="name">
                <br>
                <label for="email">Email:</label>
                <input type="email" id="email" name="email">
                <br>
                <input type="submit" value="Pošaljite">
            </form>

CSS Flexbox

CSS Flexbox je način za raspoređivanje elemenata u fleksibilnim kontejnerima:

            .flex-container {
                display: flex;
                justify-content: center;
                align-items: center;
            }

CSS Grid

CSS Grid se koristi za raspored elemenata u mreži:

            .grid-container {
                display: grid;
                grid-template-columns: 1fr 1fr 1fr;
                gap: 10px;
            }

HTML5 Semantika

HTML5 donosi semantičke tagove koji poboljšavaju strukturu stranice:

            <header>Ovo je header</header>
            <article>Ovo je članak</article>
            <footer>Ovo je footer</footer>
